Use This Tool for Free

Sharelette is a free sharing gateway. You can use it on your own website by creating a link to this page with the content you want to share as URL parameters.

  • url: The full link you want users to share.
  • text: The description that will appear in the share message.

For example, to create a "Share" link on your site, you would format your HTML like this:

<a href="https://your-website.com/sharelette.html?url=https%3A%2F%2Fcloudbreak.app&text=Check%20out%20this%20cool%20app!">
  Share this on social media
</a>

Just replace your-website.com/sharelette.html with the URL where you host this file.
